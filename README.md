# NTM Fluid Converters - HBM NTM Addon

NTM Fluid Converters is a small addon for [HBM's Nuclear Tech Mod](https://github.com/HbmMods/Hbm-s-Nuclear-Tech-GIT) that adds two blocks for bridging HBM's internal fluid system with the standard Forge fluid system. This lets you move HBM fluids in and out of any Forge-compatible tank, pipe, or machine.

Compatible with both the original HBM NTM and [JameH2's space fork](https://github.com/JameH2/Hbm-s-Nuclear-Tech-GIT).

# Features

- **HBM to Forge Converter**: Accepts fluid from HBM pipes and outputs it into any Forge `IFluidHandler` compatible block
- **Forge to HBM Converter**: Accepts fluid from any Forge `IFluidHandler` source and outputs it into HBM's fluid network

# FAQ

**Is this the official NTM?**<br>
No. This is a addon for NTM which works both with the original and space fork.

**Can I use this for my Modpack?**<br>
Absolutely!
